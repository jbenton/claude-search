How to use Claude Conversation Search

Claude doesn't let you search across your conversations — but it does let you export them. This tool lets you browse through past chats and search through them by keyword.

Getting your data

  1. Go to https://claude.ai/settings/data-privacy-controls in your browser
  2. Scroll down to the Data controls section
  3. Click the Export data button
  4. In the modal that appears, click Export data again to confirm
  5. Check your email for a download link (it might take a minute or two)
  6. Download and unzip the file you receive

Loading your conversations

  1. Click Choose File above
  2. Navigate to your unzipped folder
  3. Select the conversations.json file
  4. Your conversations will load onto this page

Using the interface

  • Sort: Toggle Last Modified/Created to change the conversation order
  • Search: Use the search box to find chat containing specific keywords
  • Browse: Click any conversation to read it in the main interface
  • Artifacts: Note that Claude's export tool doesn't include Artifacts, but in-chat code blocks will appear, and Claude's Markdown formatting (bold, code, lists) should render properly

Privacy

This tool runs entirely in your browser; your data never leaves your computer.
